# 开源AI编程openCode

复制一份opencode-cli.exe改成opencode

安装完后将openCode路径添加到环境变量

vscode添加opencode插件

然后打开项目文件夹，在terminal终端输入opencode即可打开opencode

权限配置
https://opencode.ai/docs/permissions

```json
{
  "permission": "allow"//全部允许，ask询问，deny阻止
}

{

  "permission": {
    "*": "allow",//默认允许
    "bash": "ask",//命令行询问
    "edit": "ask"//修改文件询问
  }
}
```

设置默认模型
https://opencode.ai/docs/models

```json
{
  "model": "anthropic/claude-sonnet-4-5"
}
```

忽略文件

```json
{
  "watcher": {
    "ignore": [
      "node_modules/**",
      "dist/**",
      "build/**",
      ".git/**",
      ".next/**",
      "target/**",
      "venv/**",
      "__pycache__/**",
      "*.log"
    ]
  }
}
```

默认命令工具

```json
{
  "shell": "cmd"
}
```

开启自动更新

```json
{
  "autoupdate": true
}
```
