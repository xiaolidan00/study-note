# python 镜像设置

Python 安装包国内镜像源配置全攻略（2024 最新版）
https://zhuanlan.zhihu.com/p/1896307606190024116

https://blog.csdn.net/qq_53141117/article/details/132744428

```ini
Windows系统：
1. 按 Win+R 输入 %APPDATA% 回车或者打开`C:\Users\admin\AppData\Roaming\pip`
2. 新建 pip 文件夹
3. 在pip文件夹中创建 pip.ini 文件，写入：
ini
[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
trusted-host = http://pypi.tuna.tsinghua.edu.cn
```

命令设置

```sh
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple/
pip config set install.trusted-host pypi.tuna.tsinghua.edu.cn
```

临时使用别的镜像

```sh
pip install numpy -i https://pypi.tuna.tsinghua.edu.cn/simple --trusted-host http://pypi.tuna.tsinghua.edu.cn
```

# border 和 outline

border 和 outline 很类似，但有如下区别：

outline 不占据空间，绘制于元素内容周围。
根据规范，outline 通常是矩形，但也可以是非矩形的。

```css
/* 样式 */
outline: solid;

/* 样式 | 颜色 */
outline: dashed #f66;

/* 宽度 | 样式 */
outline: thick inset;

/* 宽度 | 样式 | 颜色 */
outline: 3px solid green;
```

# 酷比魔方 i10 刷系统

## 酷比魔方官方刷机固件包

https://www.51cube.com/software/

## win 进入 bios

开机后，接入键盘，按 Esc

## 开机后按两个音量键

进入设备管理页

点击 Device Manager 可以进行跳到 USB 启动

另外可以进入 SCU 修改 Bios 设置

## 需要刷成单 win 系统

先刷 win10 32 位，将分区全部删除，再格式化作为系统盘

然后再刷官方 win8 系统

官方刷机包包含教程

可以用 rufus 制作 PE 启动，要选 NTFS，和 4096

官方 win8 系统包含触屏驱动，win10 系统没有触屏驱动，刷完会不能触屏

刷机后就没有激活了，所以能记住激活码的话尽量记住
